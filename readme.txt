----------------
Before you start
----------------

1. Set case-insensitive mode:
	git config core.ignorecase true or git config --global core.ignorecase true

2. Install LFS for Git:
	git lfs install

3. Make sure to set up in your editor (or use prepared .editorconfig file in repo):
	Indent mode - space
	Indent size - 4
	End of line - LF
	Charset - UTF-8
